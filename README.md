# pi-dotfiles

Reproducible [pi coding agent](https://github.com/badlogic/pi-mono) config.
Clones onto a fresh machine → one script → identical setup.

## What's in here

```
pi-dotfiles/
├── README.md
├── .gitignore
├── install.sh                # Linux / macOS
├── install.ps1               # Windows (PowerShell)
└── agent/                    # maps to ~/.pi/agent/
    ├── settings.json
    ├── keybindings.json
    ├── mcp.json.template     # rendered to mcp.json with ${PI_AGENT_DIR} substituted
    ├── models.json
    ├── AGENTS.md             # personal instructions for the agent
    ├── extensions/
    │   ├── claude-style-statusline.ts  # Claude-style status line above editor (model, cwd, ctx%, 5h/7d quota, ticks every 1s)
    │   ├── prompt-arrow.js       # adds ❯ prefix + indent to input editor
    │   ├── prompt-enhancer.ts    # /improve (Ctrl+Shift+E) rewrites the current prompt via LLM
    │   └── parallel-shortcut.ts  # /parallel <type1> "task" -> <type2> "task" spawns N subagents
    ├── wierd-statusline/
    │   └── events.json       # pi-statusline config (iconSet, layout, …)
    └── npm/
        ├── package.json
        └── package-lock.json # pi installs the packages listed here
```

## Restore on a new machine

### Windows (PowerShell)

```powershell
git clone <this-repo-url> pi-dotfiles
cd pi-dotfiles
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

### Linux / macOS

```bash
git clone <this-repo-url> pi-dotfiles
cd pi-dotfiles
chmod +x install.sh
./install.sh
```

Both scripts:

1. Copy `settings.json`, `models.json`, `AGENTS.md`, extension, statusline config into `~/.pi/agent/`
2. Render `mcp.json` from the template (substitutes the absolute path)
3. Run `npm install` in `~/.pi/agent/npm/` to fetch the MCP servers and tools

## After install — manual steps

The install script does **not** copy secrets. Do these by hand:

1. **Create `~/.pi/agent/auth.json`** with your provider API key:
   ```json
   { "minimax": { "type": "api_key", "key": "sk-cp-..." } }
   ```
2. **Install terminal binaries** that pi expects in `~/.pi/agent/bin/`:
   - `fd.exe` (or `fd` on Unix) — install via `choco install fd` / `apt install fd-find` / `brew install fd`
   - `rg.exe` (or `rg`) — install via `choco install ripgrep` / `apt install ripgrep` / `brew install ripgrep`
3. **Set your terminal font** to whatever you were using (Cascadia Mono by default).
   If you ever set it to a Nerd Font just for the statusline icons, you can
   revert now — `pi-statusline` here uses the `plain` icon set.
4. **Trust your projects** — pi will ask on first launch in any project folder.
   Or pre-seed `~/.pi/agent/trust.json` with your usual project paths.

## What is NOT in the repo (and why)

| Item | Reason |
|---|---|
| `auth.json` | API key — secret |
| `trust.json` | Absolute paths to your trusted projects, machine-specific |
| `sessions/` | Private conversation history |
| `tmp/` | Throwaway scratch files |
| `mcp-cache.json` | Regenerated on first run |
| `npm/node_modules/` | ~266 MB, regenerated by `npm install` |
| `bin/fd.exe`, `bin/rg.exe` | OS-specific binaries, install via package manager |

## Updating dotfiles after config changes

When you tweak settings, add an extension, or change packages:

1. Edit files under `~/.pi/agent/` as usual
2. Copy the changed files into this repo:
   ```bash
   cp ~/.pi/agent/settings.json ~/Documents/DEV/pi-dotfiles/agent/settings.json
   cp ~/.pi/agent/keybindings.json ~/Documents/DEV/pi-dotfiles/agent/keybindings.json
   cp ~/.pi/agent/extensions/*.js ~/Documents/DEV/pi-dotfiles/agent/extensions/
   cp ~/.pi/agent/extensions/*.ts ~/Documents/DEV/pi-dotfiles/agent/extensions/
   # etc.
   ```
3. `git add` + `git commit` + `git push`

## Notes on the prompt-arrow extension

`extensions/prompt-arrow.js` wraps the active editor factory and prepends
`❯ ` to the first content line (and 2-space indent to continuation lines).
It's a no-op on top of any editor that already draws its own prompt.

If the arrow renders as `?` or an empty box, your terminal font lacks
`U+276F` (heavy right-pointing angle quotation mark) — switch to a Nerd
Font or any font that includes that glyph.
